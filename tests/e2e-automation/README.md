# Z-SPEED API Automation Suite 🚀

هذا السكربت مصمم لمحاكاة دورة حياة النظام بالكامل (End-to-End) باستخدام مجموعات Postman الموجودة في المشروع. يقوم السكربت بتنفيذ الاختبارات وإنتاج تقارير احترافية جداً بصيغة HTML تحتوي على لوحة تحكم (Dashboard) كاملة.

## 📋 المتطلبات
تأكد من تثبيت المكتبات اللازمة أولاً:
```bash
npm install
```

## 🚀 كيفية التشغيل
1. تأكد من أن السيرفر يعمل:
   ```bash
   npm run start:dev
   ```
2. افتح نافذة ترمينال جديدة وشغل اختبارات النظام:
   ```bash
   npm run test:system
   ```

## 📊 التقارير (Reports)
بعد اكتمال الاختبار، ستجد التقارير في المجلد التالي:
`tests/e2e-automation/reports/`

يتم إنشاء تقرير منفصل لكل مجموعة (Collection) يحتوي على:
- **Dashboard:** ملخص شامل لعدد الطلبات الناجحة والفاشلة.
- **Total Requests:** تفاصيل كل طلب (Request) والرد (Response) الذي جاء من السيرفر.
- **Failures:** تحليل دقيق لأي خطأ حدث أثناء التست.
- **Test Scripts:** نتائج اختبارات الـ JavaScript المكتوبة داخل Postman.

## 🛠️ المجموعات التي يتم اختبارها:
1. **01 Auth & Users:** تسجيل الدخول، إنشاء الحسابات، وإدارة البروفايل.
2. **02 Restaurant & Food:** إدارة المنيو، الأقسام، وعربة التسوق.
3. **03 Orders & Drivers:** دورة حياة الطلب، تعيين السائقين، والمحافظ المالية.
4. **04 Admin & SuperAdmin:** لوحة التحكم، المراجعات، والبلاغات.
